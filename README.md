## 📌 Project Overview
This project is an **AI-powered SEO Content System for a Salon Business (AJ Salon)**.

It is designed to generate and organize:
- 📚 Blog outlines for SEO ranking
- 🧠 Long-form SEO blog content structure
- 🧩 Content clusters (pillar + supporting pages)
- 📍 Local SEO blog ideas for Hyderabad
- 🔗 Internal linking strategy for website SEO

The system simulates a **real digital marketing dashboard** used for salon growth and search engine optimization.

---

## ✨ Features

- 📊 SEO dashboard with stats overview
- 🧠 Blog outline generator (5 SEO topics)
- 📝 Long-form blog structure (1,400+ word format)
- 🧩 Content cluster strategy (pillar + supporting blogs)
- 📍 Local SEO targeting Hyderabad areas (Banjara Hills, Jubilee Hills, etc.)
- 🔗 Internal linking architecture for SEO ranking
- 🎯 Keyword-rich structure for Google ranking optimization
- ⚡ Interactive tabs (Blog / Clusters / SEO / Links)
